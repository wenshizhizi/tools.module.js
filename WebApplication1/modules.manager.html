该模块是模块加载的管理模块。

在该模块中，会定义一个全局变量【modules】。


使用模块：

<script src="scripts/modules.manager.js"></script>


引入该模块文件则会自动生成一个全局变量，由于该模块是所有其他模块的基础管理模块，因此请在加载其他模块前先引入该模块。


modules提供两个方法：


define 定义模块。
get 通过名称获取模块。



模块提供两个方法
方法名 方法详述















define(name, deps, impl)

参数

    

 {String} name 模块的名称

 {String[]} deps 该模块需要依赖的模块名称

 {Function} impl 模块实现 


如


modules.define("test", ["func", 'vers', 'tool', 'array', 'object'], function (func, vers, tool, array, object) {

    var arr = [4, 5, 7, 1, 3, 4, 6];

    array.sort(arr, function (v1, v2) {

        return v1 > v2;

    });

    

    array.each(arr, function (value,key) {

        console.log(value);


    })


    func.domReady.ready(function () {

        alert("加载完毕");

    });


    return {

        func: func,

        vers: vers,

        tool: tool

    };

});


  上面的调用中,"text"表示当前模块的名称，后面的数组表示这个模块需要依赖的其他模块名称，最后的实现将需要依赖的模块用参数的方式引入当前模块的实现。





get(name)


参数

    

 {String} name 模块名称


如


  var test = modules.get("test");


  上面的调用中，通过get获取了名称为test的模块。
